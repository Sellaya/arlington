# Public Assets

## Logo File

Please add your Arlington Estate logo image to this directory with the filename:

**`arlington-logo.png`**

### Recommended Specifications:
- **Format**: PNG (with transparency) or SVG
- **Size**: At least 200x200px for best quality
- **Aspect Ratio**: Square (1:1) recommended
- **Background**: Transparent preferred

The logo will be automatically used throughout the application:
- Sidebar header
- Mobile header
- Desktop header  
- Login page

### Alternative Formats:
If you have the logo in a different format, you can:
- Use `arlington-logo.svg` (vector format, recommended)
- Use `arlington-logo.jpg` (if no transparency needed)

The application will automatically detect and use the logo once placed in this directory.
